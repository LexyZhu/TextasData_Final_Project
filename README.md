# Paper Sieve

Paper Sieve is a web-based literature search tool that helps users efficiently collect and deduplicate academic papers across multiple databases.

## 🌐 Live Demo

https://textasdata-final-project.onrender.com/

## Features

- Search papers from arXiv, OpenAlex, Scopus, and Web of Science
- Use two groups of keywords
- Automatically deduplicate retrieved papers
- Track search progress in real time
- Export results as a CSV file
